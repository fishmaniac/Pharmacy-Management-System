## Steps to Run
### Windows
1. Open PowerShell and cd to project root directory
2. ./run.ps1
3. ./clean.ps1 (optional)
### Linux
1. Open shell and cd to project root directory
2. ./run.sh
3. ./clean.sh (optional)
